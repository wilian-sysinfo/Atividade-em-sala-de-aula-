Semana 01 — Revisão da Linguagem Python
📌 Descrição

Nesta semana foi realizada uma revisão dos principais conceitos da linguagem Python, com o objetivo de reforçar fundamentos importantes que serão utilizados ao longo do curso Técnico em Inteligência Artificial do SENAC.

📚 Conteúdos Revisados

Sintaxe básica da linguagem Python

Variáveis e tipos de dados

Estruturas condicionais (if, else, elif)

Estruturas de repetição (for, while)

Entrada e saída de dados

Operadores lógicos e aritméticos

🧠 Objetivo da Atividade

Revisar conceitos fundamentais da linguagem Python para garantir uma base sólida para o desenvolvimento das próximas atividades relacionadas à programação e à Inteligência Artificial.
