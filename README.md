# 📚 Atividades em Sala de Aula — Técnico em Inteligência Artificial (SENAC)

## 📌 Sobre o Repositório

Este repositório foi criado para armazenar, organizar e versionar as atividades desenvolvidas em sala de aula durante o curso **Técnico em Inteligência Artificial** do **SENAC**.

Os conteúdos incluem exercícios práticos, exemplos de código e atividades propostas ao longo das aulas, com o objetivo de registrar a evolução do aprendizado e a aplicação dos conceitos estudados durante a formação.

## 🎯 Objetivos

* Registrar as atividades realizadas em sala de aula
* Organizar exercícios e práticas desenvolvidas durante o curso
* Acompanhar a evolução do aprendizado ao longo das aulas
* Utilizar controle de versão para gerenciamento das atividades

## 📂 Estrutura do Repositório

As atividades são organizadas por semanas para facilitar a navegação e o acompanhamento do conteúdo trabalhado em aula.

```
📦 atividades-ia-senac
 ┣ 📂 semana01
 ┃ ┗ 📄 atividade
 ┣ 📂 semana02
 ┃ ┗ 📄 atividade
 ┣ 📂 semana03
 ┃ ┗ 📄 atividade
 ┗ 📄 README.md
```

## 🗓️ Atualizações

Este repositório será atualizado **semanalmente**, conforme novas atividades forem desenvolvidas durante as aulas.

## 🛠️ Tecnologias e Ferramentas

As tecnologias utilizadas podem variar de acordo com o conteúdo abordado durante o curso, podendo incluir:

* Python
* Ferramentas de desenvolvimento
* Bibliotecas para análise de dados
* Conceitos e práticas de Inteligência Artificial
* Controle de versão com Git e GitHub
